# Project skills

These skills are installed from [ComposioHQ/awesome-claude-skills](https://github.com/ComposioHQ/awesome-claude-skills) and are available to the Cursor agent in this project.

Each subfolder contains a `SKILL.md` that describes when and how to use the skill. Skills are loaded automatically by Cursor when relevant to your task.

To update: re-clone the repo and copy updated skill folders here, or edit individual `SKILL.md` files as needed.
